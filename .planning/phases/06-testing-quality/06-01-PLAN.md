  <action>
    Create 5 test files in `apps/web/src/lib/ai/__tests__/`. No mocking of external services needed -- all functions under test are pure.

    **router.test.ts** (test getModelForTask, getProviderForTask):
    - Each of the 8 TaskType values returns the correct model ID from MODEL_MAP
    - getProviderForTask returns 'anthropic' for plan_generation, introduction, intent_analysis
    - getProviderForTask returns 'openai' for content_editing, grammar_check, semantic_optimization, meta_generation, media_suggestions

    **context-builder.test.ts** (test buildPromptContext, buildPrompt, buildSystemMessage):
    - buildPromptContext: filters is_to_avoid terms into termsToAvoid, maps non-avoided terms with display_term/min/max/importance, uses structural_benchmarks from serpAnalysis or empty object when null, passes through userContext
    - buildPrompt: replaces {keyword}, {language}, {semantic_terms}, {terms_to_avoid}, {content}, {selected_text}, {audience}, {tone}, {sector}, {brief} in template. Unknown variables become empty string. Top 20 terms sorted by importance.
    - buildSystemMessage: includes taskType as role, keyword, language, top 10 terms, terms to avoid, audience, tone. Over-optimization warning always present.
    - Edge cases: empty semanticTerms array, null serpAnalysis, missing userContext fields

    **executor.test.ts** (test estimateCost only -- executePrompt requires AI SDK mocking which is in Plan 02):
    - Known model: estimateCost('openai/gpt-4o-mini', { promptTokens: 1000, completionTokens: 500 }) = (1000*0.15/1M) + (500*0.6/1M) = 0.00045
    - Known model: estimateCost('anthropic/claude-sonnet-4-5-20250929', { promptTokens: 1000, completionTokens: 500 }) = (1000*3/1M) + (500*15/1M) = 0.0105
    - Unknown model: returns 0
    - Zero tokens: returns 0

    **json-extractor.test.ts** (test extractJSON):
    - Strategy 1: clean JSON string -> parses directly
    - Strategy 2: markdown-wrapped JSON (```json {...} ```) -> extracts and parses
    - Strategy 3: JSON embedded in text ("Here is the result: {...}") -> finds and parses
    - Error: no valid JSON -> throws "No valid JSON found"
    - Edge: nested objects, arrays, strings with special characters

    **outline-builder.test.ts** (test buildOutlinePrompt, validateOutlineHierarchy, parseOutlineResponse):
    - buildOutlinePrompt: includes keyword, competitor headings when > 2 headings, falls back to titles when <= 2, includes topTerms numbered list
    - validateOutlineHierarchy: returns false for empty array, false if first item is h3, false if h3 before any h2, true for valid h2-then-h3 sequence
    - parseOutlineResponse: parses clean JSON array, strips markdown code fences, validates level is 'h2' or 'h3', validates title is non-empty string, validates keywords is array of strings, throws on invalid JSON/missing fields/wrong types

    Run: `cd apps/web && npx vitest run src/lib/ai/__tests__/`

    - normalizeText: handles NFD decomposition (e.g., "cafe" with accent -> "cafe" without)
    - countOccurrences: counts non-overlapping word-boundary matches (e.g., "the cat sat on the mat" for "the" -> 2)
    - countOccurrences: respects word boundaries -- "energy" should NOT match inside "bioenergy"
    - countOccurrences: handles punctuation as word boundary ("energy." counts "energy")
    - countOccurrences: returns 0 for empty term or empty text

    **scoring.test.ts** (test calculateScore, getScoreLabel, getScoreColor, calculateStructuralMetrics):
    - getScoreLabel: 0-30="Mauvais", 31-55="Moyen", 56-75="Bon", 76-100="Excellent", 101+="Sur-optimise"

    - calculateScore: score capped at 120
